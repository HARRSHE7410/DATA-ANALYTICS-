Objective:-
The objective of this task was to perform a deep-dive analysis on customer behavior and build an interactive BI dashboard to surface key business insights.

KPIs Defined:-
Conversion Rate, Customer Churn Rate, Average Order Value, and Customer Lifetime Value were identified as core KPIs due to their direct impact on revenue and retention.

Deep-Dive Analysis:-
Funnel analysis was used to identify drop-offs in the customer journey.
Cohort analysis highlighted trends in conversion by signup period.
Segmentation analysis revealed revenue differences across customer value groups.

Dashboard:-
An interactive GUI-based dashboard was developed using Streamlit, allowing users to filter customer segments and dynamically explore KPIs and trends.
